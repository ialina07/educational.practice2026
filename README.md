Студент: Иванова Алина

Научный руководитель: Кандидат технических наук, доцент кафедры системного программирования математико-механического факультета СПбГУ Литвинов Юрий Викторович

## Структура проекта
src/ - исходный код

tests/ - модульные и интеграционные тесты

bin/ - собранные бинарные файлы

obj/ - объектные файлы

## Модули
bitIo - побитовый ввод/вывод

priorityQueue - очередь с приоритетами (min-heap)

huffmanTree - построение дерева и кодов Хаффмана

compress - сжатие файла

decompress - распаковка файла

## Сборка
make

## Сжатие
./bin/huffman -c input.txt output.huff

## Распаковка
./bin/huffman -x output.huff output.txt

## Тестирование
* make testAll - запуск всех тестов
* make testBitIo - тестирование в побитового ввода/вывода
* make testPq - тестирование очереди с приоритетами
* make testCompress - тестирование сжатия
* make testDecompress - тестирование распаковки
